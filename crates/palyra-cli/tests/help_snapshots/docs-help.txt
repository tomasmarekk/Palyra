Discover bundled or committed operator docs and CLI help snapshots locally

Usage: palyra docs [OPTIONS] <COMMAND>

Commands:
  list
  search
  show

Options:
  -h, --help
          Print help (see a summary with '-h')

Global Options:
      --profile <PROFILE>

      --config <CONFIG_PATH>

      --state-root <STATE_ROOT>

  -v, --verbose...
          Increase logging verbosity (-v => debug, -vv => trace)

      --log-level <LOG_LEVEL>
          [default: info]
          [possible values: error, warn, info, debug, trace]

      --output-format <OUTPUT_FORMAT>
          [default: text]
          [possible values: text, json, ndjson]

      --plain

      --no-color

Examples:
  palyra docs list
  palyra docs search acp
  palyra docs show cli-v1-acp-shim
  palyra docs show docs/architecture/browser-service-v1.md

Discoverability:
  `docs` indexes committed source docs in-repo and bundled docs/help snapshots from portable installs for local, offline lookup.
