Discover local CLI help snapshots from source checkouts or portable installs

Usage: palyra docs [OPTIONS] <COMMAND>

Commands:
  list
  search
  show

Options:
  -h, --help
          Print help (see a summary with '-h')

Global Options:
      --profile <PROFILE>

      --expect-profile <EXPECT_PROFILE>

      --config <CONFIG_PATH>

      --state-root <STATE_ROOT>

  -v, --verbose...
          Increase logging verbosity (-v => debug, -vv => trace)

      --log-level <LOG_LEVEL>
          [default: info]
          [possible values: error, warn, info, debug, trace]

      --output-format <OUTPUT_FORMAT>
          [default: text]
          [possible values: text, json, ndjson]

      --plain

      --no-color

      --allow-profile-mismatch

      --allow-strict-profile-actions

Examples:
  palyra docs list
  palyra docs search gateway
  palyra docs search browser
  palyra docs show help/docs-help

Discoverability:
  `docs` indexes committed CLI help snapshots in source checkouts and bundled help snapshots from portable installs for local, offline lookup.
