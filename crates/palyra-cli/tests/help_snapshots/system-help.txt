Inspect runtime heartbeat, subsystem presence, and recent system events

Usage: palyra system [OPTIONS] <COMMAND>

Commands:
  heartbeat
  presence
  event      [aliases: events]

Options:
  -h, --help
          Print help (see a summary with '-h')

Global Options:
      --profile <PROFILE>

      --config <CONFIG_PATH>

      --state-root <STATE_ROOT>

  -v, --verbose...
          Increase logging verbosity (-v => debug, -vv => trace)

      --log-level <LOG_LEVEL>
          [default: info]
          [possible values: error, warn, info, debug, trace]

      --output-format <OUTPUT_FORMAT>
          [default: text]
          [possible values: text, json, ndjson]

      --plain

      --no-color

Examples:
  palyra system heartbeat
  palyra system presence --json
  palyra system event --limit 50

Discoverability:
  `system` is the top-level operator view over runtime heartbeat, subsystem presence, and recent journal events.
