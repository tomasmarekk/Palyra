Show transport and admin status across HTTP/gRPC surfaces

Usage: palyra status [OPTIONS]

Options:
      --url <URL>
      --grpc-url <GRPC_URL>
      --admin
      --token <TOKEN>
      --principal <PRINCIPAL>
      --device-id <DEVICE_ID>
      --channel <CHANNEL>
  -h, --help                   Print help

Global Options:
      --profile <PROFILE>
      --config <CONFIG_PATH>
      --state-root <STATE_ROOT>
  -v, --verbose...                     Increase logging verbosity (-v => debug, -vv => trace)
      --log-level <LOG_LEVEL>          [default: info] [possible values: error, warn, info, debug, trace]
      --output-format <OUTPUT_FORMAT>  [default: text] [possible values: text, json, ndjson]
      --plain
      --no-color
