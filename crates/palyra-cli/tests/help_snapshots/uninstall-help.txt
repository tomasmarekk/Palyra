Remove an installed Palyra package and optional state

Usage: palyra uninstall [OPTIONS]

Options:
      --install-root <INSTALL_ROOT>

      --remove-state

      --yes

      --dry-run

  -h, --help
          Print help (see a summary with '-h')

Global Options:
      --profile <PROFILE>

      --config <CONFIG_PATH>

      --state-root <STATE_ROOT>

  -v, --verbose...
          Increase logging verbosity (-v => debug, -vv => trace)

      --log-level <LOG_LEVEL>
          [default: info]
          [possible values: error, warn, info, debug, trace]

      --output-format <OUTPUT_FORMAT>
          [default: text]
          [possible values: text, json, ndjson]

      --plain

      --no-color

Examples:
  palyra uninstall --install-root ./install --dry-run
  palyra uninstall --install-root ./install --remove-state --yes
