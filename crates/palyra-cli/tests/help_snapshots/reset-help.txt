Reset selected local runtime scopes

Usage: palyra reset [OPTIONS] --scope <SCOPES>

Options:
      --scope <SCOPES>
          [possible values: config, state, service, workspace]

      --config-path <CONFIG_PATH>

      --workspace-root <WORKSPACE_ROOT>

      --dry-run

      --yes

  -h, --help
          Print help (see a summary with '-h')

Global Options:
      --profile <PROFILE>

      --state-root <STATE_ROOT>

  -v, --verbose...
          Increase logging verbosity (-v => debug, -vv => trace)

      --log-level <LOG_LEVEL>
          [default: info]
          [possible values: error, warn, info, debug, trace]

      --output-format <OUTPUT_FORMAT>
          [default: text]
          [possible values: text, json, ndjson]

      --plain

      --no-color

Examples:
  palyra reset --scope service --dry-run
  palyra reset --scope state --scope service --yes
  palyra reset --scope config --config-path ./palyra.toml --dry-run
