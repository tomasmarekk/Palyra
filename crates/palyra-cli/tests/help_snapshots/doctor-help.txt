Run diagnostics, repair previews, and rollback workflows

Usage: palyra doctor [OPTIONS]

Options:
      --strict
      --json
      --repair
      --dry-run
      --force
      --only <ONLY>
      --skip <SKIP>
      --rollback-run <ROLLBACK_RUN>
  -h, --help                         Print help

Global Options:
      --profile <PROFILE>
      --config <CONFIG_PATH>
      --state-root <STATE_ROOT>
  -v, --verbose...                     Increase logging verbosity (-v => debug, -vv => trace)
      --log-level <LOG_LEVEL>          [default: info] [possible values: error, warn, info, debug, trace]
      --output-format <OUTPUT_FORMAT>  [default: text] [possible values: text, json, ndjson]
      --plain
      --no-color
