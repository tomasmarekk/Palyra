Manage browser tabs for a session

Usage: palyra browser tabs [OPTIONS] <SESSION_ID> <COMMAND>

Commands:
  list    List tabs in a browser session
  open    Open a tab in a browser session
  switch  Switch the active tab in a browser session
  close   Close a tab in a browser session

Arguments:
  <SESSION_ID>

Options:
  -h, --help  Print help

Global Options:
      --profile <PROFILE>
      --config <CONFIG_PATH>
      --state-root <STATE_ROOT>
  -v, --verbose...                     Increase logging verbosity (-v => debug, -vv => trace)
      --log-level <LOG_LEVEL>          [default: info] [possible values: error, warn, info, debug, trace]
      --output-format <OUTPUT_FORMAT>  [default: text] [possible values: text, json, ndjson]
      --plain
      --no-color
