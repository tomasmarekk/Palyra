Run a narrow liveness/readiness probe across HTTP and gRPC gateway surfaces

Usage: palyra health [OPTIONS]

Options:
      --url <URL>

      --grpc-url <GRPC_URL>

      --json

  -h, --help
          Print help (see a summary with '-h')

Global Options:
      --profile <PROFILE>

      --expect-profile <EXPECT_PROFILE>

      --config <CONFIG_PATH>

      --state-root <STATE_ROOT>

  -v, --verbose...
          Increase logging verbosity (-v => debug, -vv => trace)

      --log-level <LOG_LEVEL>
          [default: info]
          [possible values: error, warn, info, debug, trace]

      --output-format <OUTPUT_FORMAT>
          [default: text]
          [possible values: text, json, ndjson]

      --plain

      --no-color

      --allow-profile-mismatch

      --allow-strict-profile-actions

Examples:
  palyra health
  palyra health --output-format json
  palyra gateway health --url http://127.0.0.1:7142
