# Generated Protocol Stubs (M04)

These files are generated from `.proto` contracts in `schemas/proto/`.

- `rust/protocol_stubs.rs`
- `swift/ProtocolStubs.swift`
- `kotlin/ProtocolStubs.kt`

Regenerate:

```bash
bash scripts/protocol/generate-stubs.sh
```

Validation:

```bash
bash scripts/protocol/validate-proto.sh
bash scripts/protocol/validate-rust-stubs.sh
bash scripts/protocol/validate-swift-stubs.sh
bash scripts/protocol/validate-kotlin-stubs.sh
```
