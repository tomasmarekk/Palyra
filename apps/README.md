# Client Applications Scaffold

This directory contains client-facing application scaffolds referenced by the architecture.

## Directories

- `apps/ios`: iOS node app scaffold and Swift lint baseline.
- `apps/android`: Android node app scaffold and Kotlin lint baseline.
- `apps/desktop`: desktop companion scaffold placeholder.
- `apps/browser-extension`: local MV3 extension artifact for M37 relay workflows.
- `apps/web`: web console scaffold and web lint baseline.

M02 does not implement production clients yet. It establishes structure, coding standards,
and lint entrypoints for future milestones.
